Reference

  • join [nickname] starts the game with desired name
  • shout [message] sends message to all players
  • tell [message] tells something to players near you
  • look displays items near the player
  • world [name] creates a new world with given name
  • tp [node-name] teleport to desired world
  • tp [x],[y],[z] teleport to desired location
  • distance [node-name] displays distance from given node
  • where displays the name of world where You are world
  • heritage [content] creates a heritage ( message ) which will remain on current player location
